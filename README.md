# 🏃 霓虹跑酷 (Neon Runner)
一个使用 Python 和 Pygame 开发的赛博朋克风格跑酷小游戏。

## 📖 项目简介
玩家控制一个发光的蓝色方块，通过跳跃躲避不断涌现的霓虹障碍物。游戏包含动态星空背景、粒子拖尾特效以及霓虹辉光视觉效果。

## ✨ 游戏特色
- **赛博朋克视觉**：深色背景搭配霓虹蓝、粉、绿配色。
- **动态背景**：随机生成的星空随着跑酷速度向后划过，极具速度感。
- **粒子特效**：角色奔跑时带有炫酷的拖尾光效。
- **随机障碍**：障碍物的高度和生成间隔完全随机，保证每次游玩体验都不同。

# AI极速光轮 - 遗传进化训练

这是一个基于Python Pygame的AI进化竞速游戏，玩家控制光轮移动，同时观察AI光轮群体通过遗传算法进化的有趣过程。

## 1. 具体的游戏是干什么的

AI极速光轮是一款结合了经典光轮竞速玩法和人工智能训练的教育娱乐游戏。游戏中，玩家控制一个光轮在区域内移动，留下不可穿越的轨迹路径。同时，AI光轮群体会在相同区域内移动并相互竞争生存空间。当所有AI光轮都因碰撞而死亡时，系统会启动遗传算法，根据AI的表现选择最优秀的个体进行复制和变异，生成新一代AI群体。这个过程模拟了自然选择和进化的过程，让玩家直观地看到AI智能体如何通过进化不断提升生存能力。

## 2. 游戏界面介绍

游戏界面分为两个主要区域：

**左侧游戏区（800×600像素）：**
- 青蓝色方块：玩家控制的光轮
- 蓝绿色轨迹：玩家留下的不可穿越路径
- 粉红色方块：AI光轮
- 粉红色轨迹：AI光轮留下的路径
- 黑色背景：游戏主区域

**右侧仪表盘（380×600像素）：**
- 当前世代计数器
- 游戏速度调节显示
- 历史最高分记录
- 存活AI数量统计
- 操作提示信息
- 历代最高分趋势折线图

## 3. 特点功能

- **实时AI进化**：AI群体通过遗传算法持续进化，表现越来越好
- **双重竞技模式**：玩家与AI同时在场竞技，增加趣味性
- **数据可视化**：实时绘制历代最高分趋势图，直观展示进化成果
- **多倍速游戏**：支持1-20倍速调节，快速观察AI进化过程
- **智能碰撞检测**：精确检测边界碰撞、自身轨迹碰撞和AI间碰撞
- **自动重置机制**：碰撞后自动重置游戏状态，保持流畅体验
- **遗传算法实现**：选择、复制、变异三大进化机制完整实现

## 4. 如何训练的

游戏采用经典的遗传算法进行AI训练：

**选择机制**：当所有AI死亡时，系统会选择得分最高的AI个体作为父代
**复制机制**：将最佳个体的基因信息完全复制到新一代个体中
**变异机制**：对25%的新个体进行随机变异，改变其初始移动方向
**适应度评估**：以AI的生存时间（得分）作为适应度指标
**迭代进化**：每代AI全部死亡后自动进入下一代，持续进化

通过这种机制，AI群体逐渐学会更好的避障策略和生存技巧。
